# DataSense AI

DataSense AI is an AI-powered CSV data analysis assistant built using Python, Streamlit, Pandas, Matplotlib, and the OpenAI API.

## Project Overview

This application allows users to upload almost any CSV file and automatically receive:

- Dataset preview
- Missing value analysis
- Duplicate row detection
- Data type summary
- Unique value analysis
- Summary statistics
- Automatic visualisations
- AI-generated insights
- AI data quality report
- Dataset question-answering
- Downloadable analysis report

## Tech Stack

- Python
- Streamlit
- Pandas
- Matplotlib
- OpenAI API

## Key Features

### CSV Upload
Users can upload CSV datasets and preview the first rows of the dataset.

### Dataset Overview
The app displays rows, columns, missing values, duplicate rows, data types, unique values, and summary statistics.

### Data Quality Report
The app checks missing values, duplicates, uniqueness, and provides AI-generated cleaning suggestions.

### Visualisations
The app creates basic visualisations such as histograms, scatter plots, bar charts, and correlation matrices.

### AI Insights
Using the OpenAI API, the app generates dataset insights, suggested charts, business insights, and recommended next steps.

### Ask DataSense AI
Users can ask questions about their uploaded dataset and receive AI-assisted answers.

### Download Report
Users can download a full analysis report as a text file.
